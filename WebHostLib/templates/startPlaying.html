To get started with Archipelago, we recommend looking at the following guides:

  • The Setup Guide for installing Archipelago. This guide also does an overview of the Archipelago software. For Mac users, please follow this guide instead.
  • The YAML Basic Configuration Guide for how to obtain and use config files, which are needed to play an Archipelago game
  • The Generation Guide, for more information about generating a game, no matter if it is a single player game or a game with friends.
  • The Hosting and Connection guide for how to host and join an Archipelago game.


    • If you want to generate a game on the website, click here. If you already have a pregenerated game that you would like to host, click here instead.

    Finally, for game specific information, please look at the Supported Games page.
