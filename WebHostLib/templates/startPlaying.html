Start Playing

We recommend starting with the Setup Guide to see how to install Archipelago and have an overview of the software. For Mac users, please follow this guide instead.

To start playing a game, you'll first need a config file. To see how to get one and how to use it, please follow the YAML Basic Configuration Guide.

Next, check the Generation Guide, for more information about generating a game, no matter if it is a single player game or a game with friends. To generate a game on the website, click here

For more information about how to host a game and connect to a multiworld, please check the Hosting and Connection guide. To host a pregenerated game on the website, click here.

Finally, for game specific information, please look at the Supported Games page.
